IIFEs

立即被呼叫的函式 (Immediately Invoked Function Expression, IIFE)，

Immediately 是立即的意思，invoked 則是執行某個函式時，「執行」的意思，function expression 是一種用來建立 function 的方法，總括來說，就是用 function expression 的方式建立函式後並立即執行它。
