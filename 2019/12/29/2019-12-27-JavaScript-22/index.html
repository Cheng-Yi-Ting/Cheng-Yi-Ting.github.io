結論

箭頭函式語法

  1. 沒有參數時要有小括號。
  2. 只有一個參數時可以省略小括號。
  3. 若有兩個以上的參數要有小括號。
  4. 只有一行回傳值可以省略大括號。

箭頭函式限制

  • 函式物件中的 call 、 apply 、 bind 這三個方法，無法覆蓋箭頭函式中的 this 值。
  • 不可作為建構式使用，會在使用 new 時候拋出錯誤。
  • 箭頭函式並沒有原型（prototype）屬性。
  • 沒有一般函式有的隱藏 arguments 物件。
  • 箭頭函式不能當作 generators 使用，使用 yield 會產生錯誤。
